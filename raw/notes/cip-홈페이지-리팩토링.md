# CIP 홈페이지 리팩토링 (2026-05~07) — 남은 디벨롭 요소

cipsystem.co.kr 홈페이지 리팩토링(Vue 3 SPA) 작업에서 표면 기능은 대부분 완료. 남은 디벨롭 요소를 4갈래로 기록.

## A. 기술·품질 (데이터 불필요, 바로 가능)
1. 라우트 코드 스플리팅 — 현재 16개 뷰 전부 정적 import(한 번들 308KB) → dynamic import로 초기 로딩 개선
2. 페이지별 SEO 메타 — title 1개 고정, 라우트별 document.title/meta description/OG 태그 없음
3. 404 Not-Found 페이지 — catch-all 라우트 부재
4. 이미지 width/height 속성 (CLS 방지)
5. 페이지 전환 트랜지션 (router-view 페이드)

## B. 기능 완성 (데이터 거의 불필요)
6. 게시판 검색·카테고리 필터·페이지네이션 실동작화 (현재 UI 목업)
7. 게시판 상세 seq 라우팅 (보드당 상세 1개 고정)
8. 솔루션 3종(위험성평가·드론·360CAM) 화면 이미지 — 적합 레거시 없어 보류

## C. 데이터 필요
9. 주요실적 행 확충 + 기간/규모 컬럼
10. 협력사 로고 그리드 + IT기업 실명
11. FAQ 확충(솔루션별 카테고리·문항)
12. 채용 공고 메타(근무지·경력·급여·진행중)
13. 오시는 길 지도 좌표 정식 임베드

## D. 선택·마감
14. 블루 토큰 통합(6개 블루 → 스케일 정돈) — 전역 리팩토링
15. 홈 뉴스(가상) ↔ 게시판 뉴스 완전 통일
16. 파트너 마퀴 외부 핫링크 → 자가호스팅
17. 커밋·푸시 관리

## 참고: 이번 세션 완료분
레거시 이미지 활용, 공식 로고 적용(헤더·CI·푸터), 회사소개 7종 디자인 개선(인사말 육각형 색/인증서 실물 갤러리+라이트박스/주소·업력·대표명 정합성 등), 솔루션 실사 연결(10/13 + 라이트박스), mailto 상담폼(FAQ), 게시판 뉴스 2024~2026 최신화, 조직도 인력 정성 카드, 사업분야 CTA·솔루션칩·실적 필터/비율, WebP 최적화(4.4MB→791KB, 82% 감소), 상단 배너 오로라 애니메이션 + 깊은 청록 톤(밝은 하늘색 제거), 접근성(reduced-motion·포커스트랩·키보드), 홈 다크모드.
