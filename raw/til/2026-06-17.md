# 젠킨스 배포 후 흰 화면 → ims MCP로 서버 재시작 (raw)

## 상황
- 배포(젠킨스 등) 잡은 성공했는데 접속 시 흰 화면.
- 서버 프로세스/컨테이너가 다시 안 올라온 경우.

## 조치 흐름 (ims MCP)
- list_services → 자연어를 서비스 식별자로 매칭 (dev/prod 환경 구분 필수)
- restart_service → 재시작
  - dev: 즉시
  - prod: confirm 승인 없으면 거부 (운영은 명시 승인 후)
- get_restart_status → 15~30초 후 결과 확인
  - 컨테이너 Restarting → Started, status Success 면 복구 완료

## 메모
- "배포 성공인데 흰 화면" = 코드보다 런타임 미기동을 먼저 의심.
- 카탈로그 기반 도구 재시작 → 셸 접근 불필요, prod 승인 게이트로 실수 방지.

> 공개 레포라 내부 호스트명·service_id·서버 목록 등 식별 정보는 의도적으로 기재하지 않음.
