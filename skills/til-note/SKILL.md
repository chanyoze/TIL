# TIL 노트 작성기

사용자가 배운 내용을 `C:\study\TIL\` 폴더 안에 **날짜별 markdown 파일**로 정리하는 스킬이다.

## 동작 절차

1. **주제와 내용 파악**
   - 사용자가 배운 것의 **주제**와 **핵심 내용**을 확인한다.
   - 정보가 부족하면 한두 가지만 간단히 되묻는다. (주제가 무엇인지, 핵심이 무엇인지)

2. **저장 위치 준비**
   - `C:\study\TIL\` 폴더가 없으면 만든다.
   - 오늘 날짜로 파일명을 정한다: `C:\study\TIL\YYYY-MM-DD.md`
   - 같은 날 이미 파일이 있으면 **새 항목을 이어 붙인다(append)**. 없으면 새로 만든다.

3. **항목 작성** — 아래 템플릿을 따른다.

   ```markdown
   ## <주제>

   **배운 것**
   <한두 문장으로 핵심 요약>

   **핵심 내용**
   - <포인트 1>
   - <포인트 2>

   **예시 / 코드** (있으면)
   ```

   <코드나 예시>
   ```

   **더 알아볼 것** (있으면)
   - <후속 학습 주제>

   ---
   ```

4. **결과 보고**
   - 저장된 파일 경로와 추가한 주제를 사용자에게 알려준다.

## 작성 원칙

- 간결하게. 불필요한 군더더기 없이 핵심만.
- 한국어로 작성한다.
- 코드/예시가 없으면 해당 섹션은 생략한다.
- 같은 날 여러 번 호출되면 기존 파일 맨 아래에 새 항목을 추가한다.
