# 단어장 (wordbank)

공부할 단어/용어를 `C:\study\wordbank\wordbank.md` **한 파일에 누적**해 틈틈이 복습할 수 있게 정리하는 스킬이다.
(날짜별 학습 일지는 til-note 스킬이 담당 — 이건 "쌓아가는 단어장"이다.)

## 동작 절차

1. **단어와 설명 파악**
   - 사용자가 넣으려는 **단어(용어)**와 **설명**을 확인한다.
   - 한 번에 여러 단어를 넣을 수도 있다.
   - 설명이 없으면 사용자에게 묻거나, 잘 아는 용어라면 간단히 설명을 제안한 뒤 확인받는다.

2. **저장 위치 준비**
   - **(동기화) 작업 전 최신본 받기**: `git -C "C:\study" pull --no-edit` 를 실행한다. 실패하거나 충돌하면 자동 병합하려 하지 말고 사용자에게 알린 뒤, 로컬 기준으로 계속 진행한다.
   - `C:\study\wordbank\wordbank.md` 파일이 없으면 아래 헤더로 새로 만든다.
     ```markdown
     # 단어장 (wordbank)

     ```
   - 있으면 기존 파일을 읽어 현재 단어 목록을 파악한다.

3. **항목 추가 / 갱신**
   - 단어는 **카테고리 섹션(`## 카테고리`)** 아래에 모은다. 파일 구조 예:
     ```
     ## CI/CD
     - **CI**: 지속적 통합...
     - **CD**: 지속적 배포...
     ```
   - **새 단어**: 내용에 맞는 카테고리를 정한 뒤,
     - 해당 `## 카테고리` 섹션이 이미 있으면 그 섹션 아래에 한 줄 추가한다. (섹션 내 추가 순서대로)
     - 섹션이 없으면 파일 맨 아래에 새 섹션을 만들고 그 아래에 추가한다.
     - 사용자가 카테고리를 지정하지 않으면 적절한 카테고리를 제안해 분류한다.
   - **이미 있는 단어(중복)**: 기존 설명과 새 설명을 비교해 아래 분기로 처리한다.
     - **유의미한 차이가 있으면**(새 정보·수정·보강 등): 기존 줄을 갱신하고, 무엇이 바뀌었는지 `[before] → [after]`로 보고한다.
     - **차이가 없거나 사소하면**: 갱신하지 않고 "이미 있어 그대로 둠"이라고만 알린다. (불필요한 덮어쓰기 금지)

4. **결과 보고**
   - 추가 / 갱신 / 건너뜀(skip)을 각각 구분해 알려준다.
   - 갱신한 단어는 `[before] → [after]`를 함께 보여준다.
   - 현재 총 단어 개수를 알려준다.

5. **자동 동기화 (git push)** — 단어를 실제로 **추가하거나 갱신해 파일이 바뀐 경우에만** 수행한다. (조회만 했거나 전부 skip한 경우엔 하지 않는다.)
   ```
   git -C "C:\study" add wordbank/wordbank.md
   git -C "C:\study" commit -m "<오늘 YYYY-MM-DD> / 이찬호 / - / wordbank: <추가/갱신한 단어>"
   git -C "C:\study" push
   ```
   - push 실패 원인이 non-fast-forward(원격이 앞섬)이면 `git -C "C:\study" pull --no-edit` 후 `push`를 한 번 더 시도한다.
   - 그래도 push가 안 되면(예: 오프라인) 커밋은 로컬에 남으니, 사용자에게 "나중에 push 필요"라고만 알리고 끝낸다.

## 작성 원칙

- 설명은 한 줄로 간결하게. 핵심 정의 위주.
- 한국어로 작성하되, 단어 자체가 영어/약어면 그대로 둔다. (예: API, 멱등성)
- 같은 단어를 또 넣을 때는 **의미 있는 차이가 있을 때만** 갱신한다. 차이가 없으면 그대로 두고 덮어쓰지 않는다.
- 카테고리는 `## 이름` 형식의 섹션 헤더로 만든다. **한 단어는 하나의 카테고리에만** 둔다. 비슷한 의미의 카테고리는 기존 것으로 통일한다. (예: #개발 vs #코딩 → 하나로)
- 사용자가 "단어장 보여줘"라고 하면 파일 내용을 그대로 보여준다.
- 특정 카테고리만 보고 싶어 하면(예: "CI/CD 보여줘") 해당 섹션만 보여준다.
