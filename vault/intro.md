# 📒 노트 볼트

흩어지기 쉬운 배움을 한곳에 모아두는 **개인 지식 볼트**입니다.
왼쪽 사이드바의 **폴더 트리가 그대로 카테고리** — 옵시디언처럼 폴더로 분류해요.

## 무엇이 담겨 있나

- **📚 TIL** — 매일 배운 것 (`YYYY-MM` 월별 · `YYYY-MM-DD / 내용` 형식)
- **📖 단어장** — 분야별 용어 정리 (CI/CD · Java · AI · 프론트엔드 …)
- **🏢 회사 · 🧪 토이프로젝트** — 주제별 노트 (폴더로 자유롭게 확장)

## 어떻게 쌓이나

`vault/` 아래 폴더에 `.md`를 넣으면 **자동으로** 사이드바 트리에 나타납니다. 카테고리 라벨·순서는 폴더의 `_category_.json`으로 조정해요.

> 새 기록은 `writedown` 스킬로 **raw(원본 보관) + readable(이 사이트용)** 두 버전으로 남깁니다.
