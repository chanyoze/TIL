# 📒 노트 볼트

모든 기록을 한곳에 모은 개인 지식 볼트입니다. **왼쪽 사이드바가 폴더 트리(카테고리)** 예요.

- **TIL** — 매일 배운 것 (날짜별)
- **단어장** — 용어 정리
- **회사 / …** — 주제별 카테고리 (폴더로 자유롭게 추가)

> 새 노트는 `vault/` 아래에 폴더를 만들고 `.md`를 넣으면 **자동으로** 트리에 나타납니다(옵시디언처럼). 카테고리 이름·순서는 폴더의 `_category_.json`으로 조정해요.
