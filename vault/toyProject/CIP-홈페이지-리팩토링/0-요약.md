## 1. 한눈에 보기 (요약)

| 항목 | 현황 |
|---|---|
| 백엔드 | Spring Boot **3.5.14**, Java **21**, Maven |
| 뷰 엔진 | **Thymeleaf** (서버사이드 렌더링) |
| 사내 공통모듈 | `com.cloudlab.wt:cloudlab-wt-core:2.3.1` (GitHub Packages) — 보안/JWT/EnvVariables 제공 |
| DB | Oracle (thin), 듀얼 데이터소스 `base`/`ctalk`, HikariCP |
| ORM | MyBatis 의존성 **있음** — 그러나 **Mapper XML·Service·Domain 계층 전혀 없음** |
| 현재 동작 | 컨트롤러가 **정적 HTML 뷰만 반환** (DB 연동 0건) |
| 디자인 | 레거시 classic ASP를 **1:1 복사**한 `<table>` 레이아웃, 고정폭 1180px, 인라인 스타일, 반응형 없음 |
| 포트 | 10040 |

**핵심 진단:** 지금은 "**ASP → Thymeleaf 껍데기 이전**" 단계입니다. 화면은 옮겨왔지만,
게시판 글 목록·상세가 전부 **HTML에 하드코딩**되어 있고 DB·비즈니스 로직이 없습니다.
즉 *마크업 마이그레이션은 진행 중, 기능 마이그레이션은 미착수* 상태입니다.
