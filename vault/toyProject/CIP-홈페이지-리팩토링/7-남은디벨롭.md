# 남은 디벨롭 요소 (2026-06-21 기준)

cipsystem.co.kr 홈페이지 리팩토링(Spring Boot + Thymeleaf → **Vue 3 SPA**)에서 디자인·표면 기능은 대부분 완료된 상태다. 아래는 추가로 손볼 거리를 **성격별·우선순위별**로 묶은 것. (앞선 0~6 문서의 후속 — 실제 작업이 진행되며 누적된 잔여 항목)

## A. 기술·품질 — 데이터 불필요, 바로 가능

외부 자료 없이 효과가 큰 기본기 항목들.

- **라우트 코드 스플리팅** — 현재 16개 뷰가 전부 정적 `import`라 한 번들(약 308KB)에 모두 포함. `() => import()` 동적 임포트로 초기 로딩 개선.
- **페이지별 SEO 메타** — `<title>`이 하나로 고정. 라우트별 `document.title` · meta description · OG(Open Graph) 태그 없음 → 검색·공유 불리.
- **404 Not-Found 페이지** — catch-all 라우트(`/:pathMatch(.*)*`) 부재 → 잘못된 주소는 빈 화면.
- **이미지 `width`/`height` 속성** — CLS(Cumulative Layout Shift, 로딩 중 레이아웃 밀림) 방지.
- **페이지 전환 트랜지션** — `<router-view>` 페이드 등.

## B. 기능 완성 — 데이터 거의 불필요

- **게시판 검색·카테고리 필터·페이지네이션 실동작화** — 현재 UI만 있고 동작하지 않는 목업. 클라이언트 측 필터로 살릴 수 있음.
- **게시판 상세 seq 라우팅** — 지금은 보드당 상세 글 1개 고정.
- **솔루션 3종 화면 이미지**(위험성평가·드론·360CAM) — 적합 레거시 이미지 없어 보류(스톡/제작 필요).

## C. 데이터 필요 — 실제 자료 확보 후

- 주요실적 **행 확충 + 기간/규모 컬럼**
- 협력사 **로고 그리드** + IT기업 **실명**
- **FAQ 확충**(솔루션별 카테고리·문항)
- **채용 공고 메타**(근무지·경력·급여·진행 여부)
- 오시는 길 **지도 좌표 정식 임베드**

## D. 선택·마감

- **블루 토큰 통합** — 6개 블루(`#007a9f`·`#1d6fb8`·`#005f7c`·`#0d3b66`·`#36c5f0`·`#6db6ff`)가 난립 → 스케일로 정돈(전역 리팩토링이라 신중히).
- 홈 뉴스(가상) ↔ 게시판 뉴스 **완전 통일**
- 파트너 마퀴 **외부 핫링크 → 자가호스팅**
- 커밋·푸시 관리
