# 🚧 준비중 (스테이징)

아직 공개 정리가 끝나지 않은 **초안·작업중 문서**를 임시로 모아두는 영역이다.

- **잠금**: 사이드바 하단에 분리되고, 비밀번호 게이트(🔒)로 화면에서 가려진다.
- ⚠️ GitHub 레포는 public이라 원본 `.md`는 그대로 올라간다. 이 영역은 "사이트 화면에서의 가림"만 제공한다 — **민감한 내용은 애초에 올리지 않는다.**
- 정리가 끝난 문서는 알맞은 카테고리(개발노트·토이프로젝트·회사 등)로 **옮긴다**.

> writedown 스킬에서 기록할 때 **"그냥 올림(잠금)"** 을 고르면 이 영역(또는 다른 잠금 카테고리)으로 들어간다.
