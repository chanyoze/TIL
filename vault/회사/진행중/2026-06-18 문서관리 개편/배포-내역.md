# 문서관리 / 위험성평가 모듈 배포 내역

문서관리 개편 배포에서 **개발 환경에는 있으나 운영 환경에 없는** 객체를 추려 정리한 기록이다.
[배포 체크리스트](배포-체크리스트)의 1·2번(테이블·시퀀스)을 실제 대상으로 구체화했다.
(서버·계정 등 환경 식별정보는 제외, 제품 내부 식별자만 기록)

## 비교 기준

환경 간 객체를 맞출 때 쓴 키. **PK/시퀀스로 발번되는 숫자 ID는 개발·운영이 서로 달라 비교 기준으로 쓰면 안 된다.**

| 대상 | 비교 키 |
|------|---------|
| 테이블 | `table_name` |
| 시퀀스 | `sequence_name` |
| 프로그램 / 시스템메뉴 | `SST_UNQ_CD` (프로그램 `UP…`, 메뉴 `UM…`) — 숫자 PK 비교 금지 |

## 1. 테이블 생성

`DCMT` 포함 테이블 중 개발에만 있는 것에서 **`DTDB_`(이력/백업) · `_TEMP`/`_TEMP2` · `_OPTION`** 을 제외한 4개가 생성 대상이다.

- `TCCMS_DCMT_CTGRP_CYCLE`
- `TDCMT_CTGRP_CYCLE`
- `TDCMT_DCMT_SHAREPROJ`
- `TSST_DCMT_CTGRP_CYCLE`

## 2. 시퀀스 생성

DCMT 본 시퀀스(`SDCMT_CTGR`, `SDCMT_DCMT`, `SDCMT_DCMT_VS`, `SDCMT_RFRC`)는 양쪽 동일해 생성분이 없다.
개발 전용 시퀀스가 다수 있었으나 정산·조합 등 **타 모듈 레거시라 모두 제외**하고, 이번 신규 테이블과 직접 대응되는 것만 남겼다.

- `SSST_CTGR_CYC` — 신규 `TSST_DCMT_CTGRP_CYCLE`용

## 3~7. 이후 단계

| 단계 | 메모 |
|------|------|
| 3. 공통팝업 배포 | 직접 진행 |
| 4. 프로그램 배포 | `SST_UNQ_CD` 기준 비교 시 **위험성평가 프로그램은 운영에 이미 전부 존재**(신규 배포분 없음) |
| 5. 시스템메뉴 배포 | `TCC_SYSTEM_MENU`, `SST_UNQ_CD` 기준 비교 |
| 6. 리포트 배포 | 데이터 · 파일 |
| 7. 백업테이블 생성 | 배포 전 운영 대상 백업 |
