## 배포 과정 (순서대로)

아래 대상은 **개발 환경에는 있으나 운영 환경에 없는** 객체를 차집합으로 추려 구체화한 것이다.
(서버·계정 등 환경 식별정보는 제외, 제품 내부 식별자만 기록)

### 비교 기준

환경 간 객체를 맞출 때 쓴 키. **PK/시퀀스로 발번되는 숫자 ID는 개발·운영이 서로 달라 비교 기준으로 쓰면 안 된다.**

| 대상 | 비교 키 |
|------|---------|
| 테이블 | `table_name` |
| 시퀀스 | `sequence_name` |
| 프로그램 / 시스템메뉴 | `SST_UNQ_CD` (프로그램 `UP…`, 메뉴 `UM…`) — 숫자 PK 비교 금지 |

### 단계

- [ ] 1. **테이블 생성** — 대상 4개
  - `TCCMS_DCMT_CTGRP_CYCLE` · `TDCMT_CTGRP_CYCLE` · `TDCMT_DCMT_SHAREPROJ` · `TSST_DCMT_CTGRP_CYCLE`
  - `DCMT` 포함 테이블 중 개발에만 있는 것에서 **`DTDB_`(이력/백업) · `_TEMP`/`_TEMP2` · `_OPTION`** 을 제외한 결과
- [ ] 2. **시퀀스 생성** — `SSST_CTGR_CYC` (신규 `TSST_DCMT_CTGRP_CYCLE`용)
  - DCMT 본 시퀀스(`SDCMT_CTGR`, `SDCMT_DCMT`, `SDCMT_DCMT_VS`, `SDCMT_RFRC`)는 양쪽 동일해 생성분 없음
- [ ] 3. ~~**리포트 배포** — 데이터 · 파일~~ *(금번 대상 없음)*
- [ ] 4. **백업테이블 생성** — 배포 전 운영 대상 백업
  - 위치: `서비스운영 > 개발지원 > Generator > Oracle Backup Table`
- [ ] 5. **운영서버 데이터 배포** — 아래 항목은 `서비스운영 > 개발지원 > 데이터배포 > 운영서버데이터배포`에서 일괄 처리
  - [ ] 공통팝업 배포
  - [ ] 프로그램 배포
  - [ ] 시스템메뉴 배포 — `TCC_SYSTEM_MENU`, `SST_UNQ_CD` 기준 비교
  - [ ] 위험성평가 모듈 메뉴 배포
